# Artifacts

## Crates / source of truth

- `nod-proto/`: wire DTOs (`Request`, `RequestOption`, `OptionKind`, `CardField`, `CardLink`, `Decision`, …) + canonical signing crypto + frozen protocol-freeze vectors. The contract every other surface tracks. Now also carries inert `#[typeshare]` annotations.
- `nod-proto-ffi/`: UniFFI crate (`crate-type` lib/staticlib/cdylib) exposing `request_digest`/`decision_signing_payload`/`verify_payload`/`validate_public_key` to Swift. Has an in-crate `uniffi-bindgen` bin for version-locked codegen.
- `server/nod-apns-relay/`: APNs relay as a library + thin binary. `ApnsDelivery` trait, `RelayPolicy` (bundle-id pinning + `sanitize`), `AppleApnsProvider`.
- `server/nod-server/crates/nod-server/`: axum server. `push.rs` (`PushRoute`, registry), `apns_relay.rs` (`ApnsRelayProvider` remote + `InProcessApnsProvider`), `config.rs` (`ApnsDirectConfig`).
- `client/nod-client-core/`: shared Rust client; `models.rs` holds the view types (`ClientState`, `Channel`, `ServerProfile`, `UserDevice`) serialized to the desktop frontend (now `#[typeshare]`-annotated).
- `client/nod-tui/`: Rust TUI client.
- `client/nod-desktop/`: Tauri desktop — `src-tauri/` (Rust, uses nod-client-core) + `src/` (React/TS frontend).
- `client/nod-apple/`: Swift `NodKit` library + `NodMac` + `NodIOS`. `NodSigningKeyStore.swift` signs via Secure Enclave + `NodProtoFFI`.

## Scripts

- `scripts/generate-types.sh`: typeshare TS generation (nod-proto + nod-client-core) → `client/nod-desktop/src/dto/generated.ts`. (Swift generation intentionally dropped.)
- `client/nod-apple/scripts/build-nod-proto-ffi.sh`: builds the 5 Apple static-lib slices, generates UniFFI Swift, lipo's universal slices, assembles the xcframework, places the Swift wrapper.
- `client/nod-apple/scripts/build-macos-app`: canonical NodMac `.app` bundle build (xcodebuild).

## Generated / built (not the source of truth)

- `client/nod-desktop/src/dto/generated.ts`: typeshare TS projection. Present, **currently untracked** (`??`). `models.ts` re-exports it.
- `client/nod-apple/Frameworks/nod_proto_ffiFFI.xcframework`: ~140 MB universal static-lib xcframework. **Gitignored**; rebuild via `build-nod-proto-ffi.sh`.
- `client/nod-apple/Sources/NodProtoFFI/nod_proto_ffi.swift`: generated UniFFI Swift wrapper. **Gitignored**; regenerated by the build script.
