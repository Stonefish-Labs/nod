## Goal

The current taskbar icon on MacOS is a bell with a slash, or a bell. Ideally, it should be the logo in our icon, a little character head with the sound waves or just the character head. a custom icon will go a long way in making this app feel more polished.

## Acceptance Criteria

- [x] MacOS app has a custom taskbar icon that works on light and dark mode.
- [x] Windows desktop tray uses the custom Nod head-with-waves icon.

## Notes

- Scope expanded during planning to cover both macOS and Windows.
- Use the Nod head plus sound waves, not the full rounded-square app icon.
